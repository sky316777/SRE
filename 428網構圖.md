## 改 ip_forward  
sudo nano /etc/sysctl.conf  
加一行  
net.ipv4.ip_forward=1  
再輸入  
sysctl -p  


---

## 改路由表  
sudo /etc/local.d/route_set.start  
加上要加的路由  
sudo route add -net 目的地 netmask 網路遮罩 gw 它家門  
再輸入  
sudo chmod +x /etc/local.d/route_set.start  
sudo rc-update add local
